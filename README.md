# HSSFLDON: Hybrid Semi-Supervised Federated Learning with Distributed Oracle Networks

Welcome to the repository for my [COMP6970 - Federated Learning](https://jackqqwang.github.io/26spring.html "Class Information") course project! The goal of my project is to research and explore a unique federated learning environment where unlabeled data is located at a central server, small heterogenous data is located at individual clients, and both active and passive learning are used to produce a global model!
